### 完成工作

将FPGA直出的UDP数据包转换为TCP数据包并转发到目的端口

### 技术要点

- C
- Linux API
- EPOLL模型
- 协议转发
- POSIX规范
- Ubuntu Core系统
- NanoPi R2S

### 问题汇总

- 彻底抛弃Select方式改用Epoll模型
- 建立适应高并发消息队列的内存管理
- Ubuntu18.04中开机免密登录不同于以往版本
- WSL中建立编译环境并使用VSCode进行远程调试
